description: Implement the next task incrementally — build, test, verify, commit
---

Load and follow **`.cursor/skills/incremental-implementation.md`** and **`.cursor/skills/test-driven-development.md`** (see also **`.cursor/skills/README.md`**).

Pick the next pending task from the plan. For each task:

1. Read the task's acceptance criteria
2. Load relevant context (existing code, patterns, types)
3. Write a failing test for the expected behavior (RED)
4. Implement the minimum code to pass the test (GREEN)
5. Run the full test suite to check for regressions
6. Run the build to verify compilation
7. Commit with a descriptive message
8. Mark the task complete and move to the next one

If any step fails, follow **`.cursor/skills/debugging-and-error-recovery.md`**.
