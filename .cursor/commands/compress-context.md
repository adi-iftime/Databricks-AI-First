Reduce noise while keeping what still matters for engineering work.

**Keep:** Stable decisions, interfaces/contracts, current system state, active constraints.

**Remove:** Failed experiments, long debug trails, repetitive history, abandoned ideas.

**Rules:** Do not narrate the whole conversation; do not invent new design decisions. Return only a tight working snapshot and unresolved issues if any.
